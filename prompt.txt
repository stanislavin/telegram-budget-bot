I will provide you with an expense description in Russian. The description will specify what the money was spent on and the amount spent. It may optionally include the currency. Supported currencies are: RSD, EUR, RUB. If the currency is not specified or unclear, default to RUB. It may optionally include category.

Your task is to:
1. Extract the numeric amount spent from the description.
2. Determine the currency (RSD, EUR, RUB), defaulting to RUB if unspecified or ambiguous.
3. Take the category from the description if it is provided.
4. If no category is provided, categorize the expense into one of the following categories:

   - bills (includes utilities, health insurance, home rent payments)
   - car (includes fuel, repairs, car-related expenses)
   - deliveries (e.g., courier or online order shipping)
   - entertainment (e.g., movies, concerts, games)
   - essentials (e.g., groceries, basic household items)
   - health (e.g., doctor visits, medicine, excluding insurance)
   - home (e.g., furniture, repairs, decorations)
   - presents (e.g., gifts for others)
   - toys (e.g., children's toys, games for kids)
   - services (includes subscriptions like streaming or software)
   - sport (e.g., gym, sports equipment, coach)
   - transport (e.g., cab rides, carsharing, каршеринг, ride-sharing, metro, bus)
   - whatever (e.g., miscellaneous personal spending)
   - work (e.g., work-related expenses)
   - education (e.g., courses, books, tuition)
   - bonus (this is a special category and can only be specified within the description)
   - eating out (e.g. bars, restaurants, cafes)
   - extra (use only if the expense does not fit any other category)

5. Extract the original description without the expense amount, currency, and category.

6. Classify the expense into one of the following spending types:
   - need: essential living expenses (rent, groceries, utilities, health, insurance, transport to work, essentials, bills)
   - want: discretionary spending (eating out, entertainment, toys, presents, subscriptions, sports, whatever, deliveries)
   - invest: investments in future value (education, courses, savings, work tools)

Output the response in this exact format:
expense,currency,category,spending_type,original description without the expense amount and currency

Rules:
- Use only commas as separators
- Strictly follow the field order: expense, currency, category, spending_type, description
- Exclude the expense amount, currency and category from the "original description" field
- If the category is unclear, use "extra" and ensure the output still follows the format
- spending_type must be one of: need, want, invest
- Do not include any additional information in the output
- Follow the format exactly as shown in the examples below

Examples:
Input: Потратил 500 RUB на такси в городе
Output: 500,RUB,transport,need,на такси в городе

Input: 700 на булочки в категорию essentials
Output: 700,RUB,essentials,need,на булочки

Input: 1200 EUR за аренду квартиры
Output: 1200,EUR,bills,need,за аренду квартиры

Input: 300 на подарок другу
Output: 300,RUB,presents,want,на подарок другу

Input: 1500 на курс программирования
Output: 1500,RUB,education,invest,на курс программирования
